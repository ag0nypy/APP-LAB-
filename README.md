# AppLab v1

**AppLab** é um gerenciador de aplicativos moderno e intuitivo para Windows, desenvolvido para simplificar a forma como você acessa e gerencia seus programas instalados. Com uma interface inspirada no Material Design 3, você pode encontrar, executar e organizar seus atalhos com facilidade.

---

## ✨ Funcionalidades

* **🔍 Busca Inteligente:** Encontre qualquer programa instalado instantaneamente.
* **⚡ Execução Rápida:** Abra aplicativos com um único clique ou clique duplo.
* **🛡️ Modo Administrador:** Execute programas com privilégios elevados diretamente do AppLab.
* **📂 Local do Arquivo:** Abra a pasta de instalação do programa num piscar de olhos.
* **🖥️ Atalho na Área de Trabalho:** Crie atalhos no Desktop com um botão.
* **📋 Propriedades Detalhadas:** Visualize caminhos de destino, diretórios de trabalho e descrições.
* **🎨 Interface Moderna:** Desenvolvido com CustomTkinter para uma experiência *Dark Mode* premium.

## 🛠️ Tecnologias Utilizadas

* **Python** - Linguagem base.
* **CustomTkinter** - Interface gráfica moderna.
* **Pillow (PIL)** - Manipulação de ícones e imagens.
* **PyWin32** - Integração profunda com a API do Windows.
* **Winshell** - Gerenciamento avançado de atalhos.
